
## Description

This API is based to improve difetent skills:
- Migrations
- Testing
- Logging


## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## stack 
- NestJS
- TypeORM
- PostgreSQL
- Jest
